# ORGANIZADOR DE PROGRESSÃO CURRICULAR

**Ferramenta digital de planejamento docente**

DU de Langue et Cultures Brésilienne et Portugaise · Université Jean Moulin Lyon 3

---

## MENÇÃO DE AUTORIA E LICENÇA

### 1. Autoria

Esta ferramenta – código, interface, arquitetura conceitual e base de dados curricular – foi desenvolvida por **Teurra Fernandes Vailatti**, Leitora Guimarães Rosa (Instituto Guimarães Rosa / Ministério das Relações Exteriores do Brasil), no Département d'Études Lusophones da Université Jean Moulin Lyon 3, entre 2024 e 2026.

A ferramenta integra duas camadas inseparáveis: o software (HTML, CSS, JavaScript) e a arquitetura curricular (a categorização E/A/S, o sistema das seis dimensões, os eixos transversais, a base de itens com marcações temáticas e o sistema de afinidades).
